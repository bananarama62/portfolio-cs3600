tail

Tail

Team size: 1

Project Status: Complete

Replication of the tail utility commonly provided with Linux systems. Built as part of a school project. I/O operations were performed using only commands found in section 2 of the Linux man pages. The primary functions used were read(2) and write(2).

Languages Used

  • C

Systems Used

  • Make

Takeaways

This project taught me that functions such as printf(3) make coding much easier than using lower level commands like read(2) and write(2).
