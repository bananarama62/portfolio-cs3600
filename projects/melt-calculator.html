melt-calculator

Melt Calculator

Team size: 1

Project Status: Active

This melt calculator application is used to find the intrinsic value of coins made of precious metals.

Languages Used

  • Python
  • SQL

Systems Used

  • MariaDB

Takeaways

This project taught me the value of using the right tool for the job. Its first iteration featured coins from seven different countries, each stored in their own file. The data for each coin was stored natively in python, which was a bear to update the system. As the number of coins in the system increased, the headache of managing them grew. I then decided to move the storage of the coin data to a database. The major benefit of this is a much greater separation of the data from the logic of the program. It is now much easier to update the system, as I can modify the data without having to touch the logic code.

Something I would have done differently when switching over to the database would be spending more time in the planning stages. I believe that I could have split the tables better such that it would be easier to add new features in the future. I have not run into issues yet, but I believe that I might.

Future Plans

The primary focus of the project at the moment is adding more data for coins. I plan on having coin data spanning from 1850-1964 for every country. After that, I will add support for various different tags, such as one for coins still in production (namely bullion).

While the program is currently bound to the terminal, I would like to make a web based version. I plan on this program hosting a web application that could be accessed through a web browser.
