lc3-calculator

LC-3 Assembly Language Calculator

Team size: 1

Project Status: Complete

The Little Computer 3 (LC-3) is a replication of a 16 bit computer. It is programmed in their own assembly language. This project is a calculator that supports the four basic operations (+-/*) and parentheses.

Languages Used

  • LC-3 Assembly

Systems Used

  • LC-3

Takeaways

I was forced to think at a much lower level than I was used to. I had to create algorithms for performing all of the different operations. I also needed to create support for parentheses, and multiple operations in a single string. I did this through multiple arrays. Array 1 stores constants found in the expression string. Array 2 is created as the expression is parsed. Each operation pushes a frame object of the form (value pointer1, value pointer2, operation, and result). The value pointers will either point to a value in the constants array or a value in the result address of some other frame object. At the end of parsing, the second array is ran through, with operations performed and results stored.
